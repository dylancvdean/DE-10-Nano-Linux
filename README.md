# DE-10-Nano-Linux

This is designed to be a single repository that will allow you to compile the sd card image for the DE-10 Nano board.

Get started with 
<code>
./init
</code>

and
<code>
source poky/oe-init-build-env ./build
</code>

Then make any desired changes, and run
<code>
./compile
</code>
